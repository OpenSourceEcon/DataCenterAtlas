# Washington data
The State of Washington has 39 counties.

Washington state assessment ratios by county are the total assessed value of property in the county divided by the total true and fair value (fair market value). In 2025, these county assessment ratios for both real and personal property (combined) ranged from 75.1 to 96.3. See Washington Department of Revenue, "[Property Tax Statistics 2025](https://dor.wa.gov/sites/default/files/2025-10/PropTx2025_2.pdf)", Table on page 16, "Comparison of Assessment Ratios for Taxes Due in 2024 and 2025 by County," Oct. 2025.

The Washington Department of Revenue, "[Property Tax Statistics 2025](https://dor.wa.gov/sites/default/files/2025-10/PropTx2025_2.pdf)" document also has a Table on page 92 of "Comparison of Average Levy Rates by Year Due in 2018-2025. These levy rates are reported by county as dollars of total property tax liability per $1,000 of assessed value.

Our definition of average effective property tax rate by county is the total levy amount by county divided by the current use assessment value by county. The "[Property Tax Statistics 2025](https://dor.wa.gov/sites/default/files/2025-10/PropTx2025_2.pdf)" document has this table on page 91. We take these as our county average effective tax rates, and extract them directly from the spreadsheet ([`Table_27_2025.xlsx`](https://dor.wa.gov/sites/default/files/2025-10/Table_27_2025.xlsx)) associated with the table on page 91 of the PDF document. We have saved this table as [`/data/WA/Table_27_2025.xlsx`](https://github.com/OpenSourceEcon/DataCenterAtlas/blob/main/data/WA/Table_27_2025.xlsx) in the open source repository for the DataCenterAtlas.org web tool.

The average effective property tax rate data from this computation is stored as [`/data/wA/cnty_prop_tax_rates_wa_2025.csv`](https://github.com/OpenSourceEcon/DataCenterAtlas/blob/main/data/wA/cnty_prop_tax_rates_wa_2025.csv) in the open source GitHub repository for the DataCenterAtlas.org web tool. All the code that used the source data above to calculate the average effective property tax rates by county in Georgia is available as [`/code/WA/wa_analyses.py`](https://github.com/OpenSourceEcon/DataCenterAtlas/blob/main/code/WA/wa_analyses.py).
